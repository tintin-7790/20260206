# 指尖非遗：景德镇制瓷（高保真版）

一个基于Three.js的虚拟陶瓷制作小程序，允许用户在手机端通过触屏交互，体验从拉坯到烧制的完整制瓷流程。

## 核心功能

### 1. 拉坯模块：虚拟辘轳
- **交互**：屏幕中央模拟转动的辘轳与泥柱。用户单指触摸泥柱区域并上下拖动，模拟用手拉高或压低泥坯；双指捏合可控制泥柱开口大小。
- **技术实现**：
  - 使用Three.js的LatheGeometry生成动态泥坯模型
  - 实现了物理模拟，手势力度影响形变速度与平滑度
  - 添加了辘轳转动的音效
  - 当操作过快时会提示"泥坯即将塌陷"

### 2. 利坯（修坯）模块：精雕细琢
- **交互**：旋转的3D泥坯模型。用户单指触摸，虚拟"修坯刀"会刮掉接触部分的泥料，模型相应变薄、塑形。
- **技术实现**：
  - 实现了射线检测来识别触摸位置
  - 添加了泥屑飞出的粒子特效
  - 设定了厚度阈值，刮得太薄会提示"泥坯太薄了"

### 3. 上釉模块：色彩与配方
- **交互**：
  - 选色：提供经典釉色色板（如天青、梅子青、釉里红）。
  - 涂抹：选择釉色后，手指在坯体上涂抹，釉色实时覆盖。
- **技术实现**：
  - 提供了5种经典釉色
  - 实现了动态纹理贴图，支持实时绘制

### 4. 烧制模块：窑火淬炼
- **交互**：
  - 用户将上好釉的作品放入虚拟窑炉。
  - 选择烧成气氛（还原焰/氧化焰），此选择将影响最终釉色呈现。
  - 点击"点火"，观看一段烧制过程动画（温度上升、窑火变化）。
  - 开窑：最终呈现带开片、窑变等随机效果的成品。
- **技术实现**：
  - 支持选择还原焰或氧化焰
  - 实现了温度逐渐上升的动画效果
  - 烧制完成后会随机呈现成功、窑变或失败的效果
  - 根据选择的气氛和釉色生成最终的烧制颜色

## 如何部署

### 方法1：使用GitHub Pages
1. 在GitHub上创建一个新仓库
2. 将所有文件上传到仓库
3. 在仓库设置中启用GitHub Pages
4. GitHub会为您生成一个URL，您可以在手机上访问这个URL

### 方法2：使用Netlify
1. 访问 https://app.netlify.com/drop
2. 将所有文件拖放到上传区域
3. Netlify会自动部署您的项目并生成一个URL

### 方法3：使用Vercel
1. 访问 https://vercel.com/new
2. 选择"Import Project"
3. 上传您的项目文件
4. Vercel会自动部署您的项目并生成一个URL

## 技术栈
- Three.js - 3D渲染引擎
- HTML5/CSS3 - 页面结构和样式
- JavaScript - 交互逻辑和动画效果

## 注意事项
- 本项目需要互联网连接，因为它使用了CDN加载Three.js库
- 为了获得最佳体验，建议在现代浏览器中运行（如Chrome、Safari、Firefox）
- 由于使用了WebGL，某些旧设备可能无法正常运行
